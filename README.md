# FormApp

A small SwiftUI app for managing contacts. Part of my university lab assignment.

## Purpose

This project is used to practise **layouts** and **navigation** in iOS development. It includes a contact list, a form for adding/editing contacts, and a details screen with a simple, readable layout.

## Features

- Contact list with alphabetical sections
- Add and edit contacts (form with minimal validation)
- Contact details with combined address and optional fields
- SwiftUI, clean structure, minimal yet pretty design
